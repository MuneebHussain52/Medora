### ✨ Features

- 🔐 **User Authentication** — Registration & login with JWT-protected routes
- 📝 **Post Management** — Create and delete posts with ease
- ❤️ **Likes & Comments** — Engage with posts in real time
- 👥 **Follow System** — Follow and unfollow other users
- 👤 **User Profiles** — Dedicated profile pages per user
- 📱 **Responsive UI** — Clean, mobile-friendly design with Tailwind CSS
- ⚡ **Blazing Fast** — Powered by React + Vite for lightning-fast rendering

### 🚀 Getting Started

#### Prerequisites

- Node.js v16+
- MongoDB (local or Atlas)

#### Backend Setup

```bash
cd Backend
npm install
cp .env.example .env   # Add your credentials
npm start
```

#### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Open your browser at `http://localhost:5173` 🎉
